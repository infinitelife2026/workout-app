Program Guide

Everything you need to know

Client Profile

58-year-old woman, 175 lbs, intermediate fitness level

Full gym access (machines, free weights, cables, cardio)

Program Goal

Build lean muscle, improve cardiovascular health, increase mobility, and support sustainable fat loss.

3 Phases

  • Phase 1: Foundation (Weeks 1-4) — Build base strength, learn movement patterns
  • Phase 2: Building (Weeks 5-8) — Increase load, supersets, progressive overload
  • Phase 3: Peak (Weeks 9-12) — High intensity, advanced supersets, peak performance

Weekly Schedule

  • Monday: Upper Body PUSH (Chest, Shoulders, Triceps) — 45-60 min
  • Tuesday: Lower Body A (Quads, Glutes, Hamstrings) — 45-60 min
  • Wednesday: Upper Body PULL (Back, Biceps, Rear Delts) — 45-60 min
  • Thursday: Lower Body B / Glute Focus + Cardio — 45-60 min
  • Friday: Full Body HIIT + Core — 30-45 min
  • Saturday: REST or Active Recovery
  • Sunday: REST

Exercise Types

Warm-Up — 5 min general warm-up + dynamic stretches
Superset (SS) — Two exercises back-to-back, DIFFERENT muscle groups, 45-60 sec rest after both
Compound Set (CS) — Two exercises back-to-back, SAME muscle group, 60 sec rest after both
Strength — Standard straight set, 60-90 sec rest between sets
Cardio — Steady-state or interval-based conditioning
Cool-Down — Static stretching, hold 20-30 seconds

Important Tips

  • Warm up 5 min before every session + dynamic stretches
  • Cool down 5 min after with static stretching
  • Use machines over free weights when joints feel stiff
  • Rest: 45-60 sec for supersets, 60-90 sec for heavy lifts
  • If you can do 20+ reps easily, increase the weight
  • Drink at least 64 oz of water daily, more on workout days
